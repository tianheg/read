## 现代 JS 框架

它指 Lit, Solid, Svelte, Vue，处于“后 React 时代的框架”。

它们的共同特征：

- 响应式更新 DOM
- 使用克隆的 templates 用于 DOM 渲染
- 使用现代 Web APIs 让以上操作更容易
